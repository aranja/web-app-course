App Development

Part 1: HTML

Building

grunt build

Dist folder ready for deployment.

- Linting all JavaScript files against JSHint Recompiling all CoffeeScript and SASS files for production - Using r.js to compile and optimize any AMD modules Concatenation and minification of scripts and stylesheets - Compressing your images using OptiPNG for PNG files and JPEGtran-turbo for JPEGs - Running any unit tests written against a headless WebKit browser (via PhantomJS) - Creating an Application Cache manifest via Confess.js - Using revision filenames or oldernames

Deployment

Deploy dist folder to any static file server.

Distribution

Chrome Web Store

$5 developer fee.

Google gets 5%.

Host or package the game.

$5 is the fee for developer account. You can test your game without paying (and publish within your domain), but need to pay for publishing. Hosted = your own web server. Packaged = upload zip with the game.

Windows Store

$49 developer fee.

Microsoft gets 30%.

Get Visual Studio for free.

$49 for individuals and $99 for companies. The 2012 Express edition of Visual Studio is free.

App Store

$99/year developer fee.

Apple gets 30%.

Adobe PhoneGap Build

$99 for individuals and $249 for companies. With PhoneGap Build you can upload your code (or fetch from github directly) and build it for all platforms (iOS, Windows Phone, HP, Android, Blackberry and Symbian). You need to add signing key for iOS, Android and Blackberry. Phonegap Build will provide packaged apps for the platforms, ready to publish.

Play Store

$25 developer fee.

Google gets 30%.
